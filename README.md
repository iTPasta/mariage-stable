## Prérequis (court)
- Python 3.8+ avec Tkinter
- `matplotlib`

Linux (Debian/Ubuntu): `sudo apt-get install -y python3-tk && python3 -m pip install matplotlib`

macOS: Python depuis python.org (Tkinter inclus), puis `python3 -m pip install matplotlib`

Windows: Python depuis python.org (Tkinter inclus), puis `py -m pip install matplotlib`
## Lancer l’application
```bash
cd mariage-stable
python3 gui_main.py
```
